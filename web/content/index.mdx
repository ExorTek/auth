# `@exortek/auth`

A framework-agnostic authentication toolkit for Node.js ‚Äî built as 22 small packages under one scope. Pick the one you
need, or install the umbrella.

Every package is built on `node:crypto`, with **no required external dependencies** outside the `@exortek/*` scope.
The heaviest primitives (Argon2id, bcrypt) come from opt-in peer packages you install only if you use them. Pure
JavaScript with JSDoc ‚Üí `.d.ts`. Ships adapters for Express and Fastify ‚Äî the classic Node HTTP stack.

## Install

Every published package installs on its own:

```bash
npm install @exortek/crypto
npm install @exortek/security
npm install @exortek/otp
npm install @exortek/password         # + optional: argon2, bcryptjs
npm install @exortek/session          # + optional: ioredis or redis
npm install @exortek/jwk
npm install @exortek/jws
npm install @exortek/jwt              # + optional: ioredis or redis
```

Node.js **22 or newer**.
