# @exortek/auth

Framework-agnostic, zero-dependency, pure JavaScript authentication library for Node.js — a modern successor to Passport.js.

## Why

- **Zero external deps.** Everything is built on `node:crypto`.
- **Pure JavaScript with JSDoc types.** No TypeScript source — `.d.ts` is emitted from JSDoc.
- **Framework-agnostic.** Works under Express, Fastify, Hono and Elysia.
- **Opinionated defaults.** Developer writes one line, gets a safe primitive.

## Quick start

```bash
npm install @exortek/crypto
```

```js
import { hash, cipher, random, sign } from '@exortek/crypto'

const otp    = random.pin(6)                              // '847291'
const digest = hash('hello')                              // sha256 hex
const key    = await cipher.generateKey()                 // AES-256-GCM
const token  = cipher.encryptToString('state', key)       // safe for cookies

const kp  = await sign.generateSignKeyPair('es256')
const sig = sign.sign('payload', kp.privateKey, { algo: 'es256' })
```
